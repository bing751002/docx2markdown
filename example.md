# 功能說明

  -----------------------------------------------------------------------------------------------------
  **功能名稱**       廠商及合作夥伴權限與選單維護
  ------------------ ----------------------------------------------------------------------------------
  **功能代號**       SRM-TX-0311

  **功能簡述**       提供研發人員依照角色(包含一般廠商、特約商角色、外包倉角色等)設定B2B
                     選單權限，亦可依照特定廠商及合作夥伴設定B2B 選單權限。

  **進入功能路徑**   供應商暨合作夥伴平台\>廠商及合作夥伴帳號與權限維護\>廠商及合作夥伴權限與選單維護

  **使用權限控管**   ■是 □否

  **使用審核流程**   □是 ■否
