功能說明 查詢角色清單
系統/提供者 SRM / AP
執行方式 API
設計規格
API path AccessRoleInfo/QueryAccessMenuRoles
檢核邏輯
業務邏輯

  1. 呼叫AccessRoleInfoService方法QueryAccessMenuRoles

  2. 取得角色清單
